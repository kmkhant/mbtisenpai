## Quick Start

### For Developers

1. Start with [CODEBASE-REVIEW-2025.md](./reviews/CODEBASE-REVIEW-2025.md) for an overview
2. Check [CURRENT-CODEBASE-STATUS.md](./status/CURRENT-CODEBASE-STATUS.md) for current state
3. Review [FINAL-ISSUES-REVIEW.md](./issues/FINAL-ISSUES-REVIEW.md) to see what's been fixed

### For Deployment

1. Read [DEPLOYMENT-GUIDE.md](./deployment/DEPLOYMENT-GUIDE.md)
2. Configure Redis using [REDIS-SETUP.md](./deployment/REDIS-SETUP.md)

### For Understanding the System

1. Read [COMPREHENSIVE-CODEBASE-REVIEW.md](./reviews/COMPREHENSIVE-CODEBASE-REVIEW.md) for architecture
2. Review [SCORING-ANALYSIS.md](./reviews/SCORING-ANALYSIS.md) for scoring algorithm
3. Check test results in [testing/](./testing/) directory
